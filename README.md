## âš™ï¸ Required Config Vars

After clicking **Deploy to Heroku**, set these config vars:

| Key | Value |
|-----|-------|
| `BOT_TOKEN` | Your Telegram bot token |
| `API_ID` | From my.telegram.org |
| `API_HASH` | From my.telegram.org |
| `MONGO_DB_URI` | MongoDB database URI |
| `SESSION_NAME` | String session (optional) |
| `SUPPORT_CHAT` | Your support group/channel |
| `UPDATES_CHANNEL` | Your updates channel |
| `OWNER_ID` | Your Telegram ID |
| `DURATION_LIMIT` | Max song duration (ex: 120) |

> Make sure all vars are filled correctly before deploying.
